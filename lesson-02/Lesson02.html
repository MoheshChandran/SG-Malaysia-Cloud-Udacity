What is Shell

A shell is the outermost operating system (OS) layer (could be graphical or text based) that allows us to interact with the tools and services offerred by OS.

Example of text based shell (Command-Line Interface or CLI) :
  • Bash (Bourne Again Shell) used in most Linux (Bourne Again Shell),
  • Microsoft DOS (MSDOS) from Windows
  • Microsoft Powershell from Windows

There is few Unix-Styled shell for Windows OS: -
  • Git Bash - comes with git installation
  • Windows Subsystem for Linux (WSL) - a mini like VM Linux OS

Basic Bash command and concept

Here is the list of common Bash commands used:

  • echo - to print out the string typed after the command
    echo Hello World!
    Hello World!
  • variable - hold value assigned, Used $ in frontof variable to get the value set
    x=100 # Set variable x to 100
    echo 'The answer is $x.' # print out value of variable x
    The answer is 100.
  • ls - to list file or subdirectories inside the current directory
    ls # list files/directories in current directory
    ls Downloads # list files/directories in Downloads directory.
    ls . # list files/directories in current directory. Character . means current directory
    ls ~ # list files/directories in user's home directory (users's top-level directory)
  • cd - change directory. Directory name is case sensitive
    cd Downloads # change to Downloads directory
    cd .. # change to parent directory relative to current directory
  • pwd - print out current working directory/path
    cd ~
    pwd # /Users/karl
  • Relative path - give the path relative to your current location (the current working directory)
    Absolute path - provide the full path, all the way from the /(root) directory.
    pwd # /Users/karl
    cd Downloads/Photos # change to Photos directory inside Download directory
    pwd # /Users/karl/Downloads/Photos
    cd /Users/karl/Downloads/Photos # same as above using absolute path
  • Option - command argument/parameter to provide extra feature/behavior
    ls -l Document/*.pdf # Options -l to display in long format. List all pdf file in Document. Character * means any characters
    ls *poem* # List all files names that contains "poem" characters
  • mkdir - make directory
    mkdir Photos # make Photos directory inside current/working directory
  • mv - move files/directories
    mkdir Photos # make Photos directory inside current/working directory
    mv kitten.jpg Photos/kitten.jpg # move kitten.jpg into Photos directory
    mv kitten.jpg Photos/ # same as above
    mv *.jpg Photos # move all jpg files into Photos directory
  • curl - utilities command to download a file from the Web
    curl 'http://www.google.com' # display HTML contents from the url
    curl -L 'http://www.googel.com' # option -L means follow URL redirect, and download the content from the redirected url
    curl -L -o some_file.html 'http://www.google.com' # option o to save content into some_file.html
  • To view content of file, use cat (concatenate) or less commands. Use less to view big files
    cat big_file.txt # display all contents in the file
    less big_file.txt # display 1 page at a time. Use down/spacebar & up keys to scroll page down or up. Use / to search
  • rm - remove/delete files. Use option i to provide prompt feedback on the delete attempt. Use single quote for filename with space
    mkdir example # make example directory
    rm example # permanently delete example file
    rm example # permanently delete example file with prompt
    rmdir example1 example2 example3 # remove example1, example2 & example3 file
    rmdir 'example 1' # remove file with name contains space
  • rmdir - remove/delete directory
    rmdir example # delete example directory
