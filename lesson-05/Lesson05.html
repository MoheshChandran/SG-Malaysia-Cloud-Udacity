Displaying a repository's commits

  • run "git status" command

Changing how git log displays information

  1. Take a look at the output from running "git log"
    • the SHA - Each SHA is unique id for each single commit.
    • the author - The author of the commit.
    • the date - By default, the "git log" command whos the date for each commit was made.
    • the commit message - THe most important part of each single commit.
  2. Using a flag for better display of running git log command
    • The flag is "--oneline"

Viewing modified files

  • Flag: "--stat", 'stat' is short for statistics as you have already assumed.
  • Running "git log --stat" shows additional info of the number of added/removed lines

Viewng File Changes

  • Flag: "--patch",or "-p"
  • This flag is used to display the actual changes made to a file.

Viewing a specific commit

  • git shot + SHA : this shows a specific commit's content.
